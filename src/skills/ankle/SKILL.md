# Ankle Clinical Skill

## Special Tests (Pose-Based)

### Anterior Drawer Test
- **Position:** Seated, ankle at 20° plantarflexion, stabilize tibia, pull talus forward
- **Positive:** Excessive anterior translation = ATFL laxity (lateral ankle instability)
- **Keypoints:** Anterior tibiotalar displacement relative to baseline

### Talar Tilt Test
- **Position:** Seated, ankle neutral, apply inversion stress
- **Positive:** >10° tilt vs. contralateral = CFL laxity
- **Keypoints:** Inversion angle under stress vs. resting

### Thompson Test (Achilles)
- **Position:** Prone, squeeze calf
- **Positive:** No plantarflexion = Achilles rupture (EMERGENCY REFER)
- **Keypoints:** Ankle dorsi/plantarflexion angle response

### Single-Leg Balance Test
- **Position:** Single-leg stance, eyes open → eyes closed, 30 seconds each
- **Positive:** Unable to maintain > 10 seconds = proprioceptive deficit
- **Keypoints:** Center of mass sway (hip/ankle strategy), step-offs

### Weight-Bearing Lunge Test (Dorsiflexion ROM)
- **Position:** Standing lunge, knee to wall, measure distance
- **Positive:** < 10 cm knee-to-wall = dorsiflexion restriction
- **Keypoints:** Ankle dorsiflexion angle, knee-to-wall distance estimate

## Differential Diagnosis Decision Tree

```
Lateral ankle pain
├── Post-inversion injury → anterior drawer → lateral ligament sprain
├── Chronic instability → recurrent sprains → proprioceptive rehab
├── Posterior to malleolus → peroneal tendon pathology
└── Sinus tarsi tenderness → sinus tarsi syndrome

Anterior ankle pain
├── Dorsiflexion restriction → anterior impingement
├── After eversion injury → high ankle sprain (syndesmosis)
└── Gradual onset runner → anterior tibialis tendinopathy

Posterior ankle/heel pain
├── Insertional pain → Achilles insertional tendinopathy
├── Midsubstance thickening → Achilles midportion tendinopathy
├── Sudden "pop" → Achilles rupture (EMERGENCY)
└── Plantar heel → plantar fasciitis

Medial ankle pain
├── Behind medial malleolus → posterior tibialis tendinopathy
├── Flat foot + pain → adult-acquired flatfoot
└── After eversion injury → deltoid ligament sprain
```

## Exercise-to-Finding Matrix

| Finding | Primary Exercises | Contraindications |
|---------|------------------|-------------------|
| Lateral sprain (acute) | RICE, alphabet ROM, isometrics | Weight-bearing if unable, inversion |
| Chronic instability | Proprioception (BAPS board), peroneal strengthening | Unstable surfaces too early |
| Achilles tendinopathy | Eccentric heel drops (Alfredson protocol), calf raises | Stretching into pain, jumping |
| Dorsiflexion deficit | Wall lunge stretch, banded joint mobilization | Forced dorsiflexion post-fracture |
| Plantar fasciitis | Calf stretching, plantar fascia stretch, arch strengthening | Barefoot running, flat shoes |
| Proprioceptive deficit | Single-leg stance progressions, perturbation training | Unstable surfaces if acute |

## Red Flags — Ankle Specific

- **Inability to weight-bear 4+ steps** → fracture (Ottawa Ankle Rules)
- **Sudden "pop" with loss of push-off** → Achilles rupture
- **Rapidly progressive bilateral swelling** → systemic (DVT, infection)
- **Night pain + swelling in adolescent** → tumor workup
- **Foot drop after injury** → peroneal nerve injury

## Progression Criteria

| Stage | Criteria to Advance | Exercise Focus |
|-------|-------------------|----------------|
| Acute | Pain < 4/10 at rest, edema controlled | RICE, isometrics, gentle ROM |
| Subacute | Full ROM, pain-free walking | Isotonics, balance, proprioception |
| Strengthening | Single-leg balance > 30s, pain-free stairs | Progressive resistance, agility |
| Return to sport | Hop test > 90% limb symmetry | Plyometrics, sport-specific drills |
